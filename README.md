# 🧮 CodeAlpha - Calculator

## 📌 Project Overview
This project is a simple and responsive Calculator developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations with a clean, user-friendly interface and real-time calculations.

## ✨ Features
- Addition, Subtraction, Multiplication, and Division
- Clear (AC) functionality
- Delete (Backspace) option
- Responsive design for desktop and mobile devices
- Clean and modern user interface
- Instant calculation results

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure
```
CodeAlpha_Calculator/
│── index.html
│── style.css
│── script.js
│── output1.png
└── README.md
```

## 🚀 How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it in any web browser.
4. Use the calculator to perform arithmetic operations.
